Nine Packs
==========
- https://open.kattis.com/problems/ninepacks

#### Tags
- DP
- 0-1 Knapsack

#### Solution
- [c++](ninepacks.cpp)

Since the total number of hot dogs and buns <= 1e5, we can calculate the min
number of elements needed to get to a certain sum, do this for both hot dogs
and buns separately.
